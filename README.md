Project này mô phỏng cánh tay robot (robot arm) và mô phỏng BPSK trong Unity, sử dụng MATLAB để tính toán động học (kinematics) và điều khiển.
- Unity đảm nhiệm:
  + Hiển thị 3D
  + Animation
  + Tương tác người dùng
- MATLAB đảm nhiệm:
  + Tính toán Forward Kinematics (FK)
  + Inverse Kinematics (IK)
  + Điều khiển quỹ đạo
 
- Kiến trúc:  UNITY => (gửi target position) => MATLAB => (tính toán IK) => UNITY

- Yêu cầu cài đặt:
  + Cài đặt môi trường MATLAB: MATLAB R2021a
  + Chạy bản build exe
